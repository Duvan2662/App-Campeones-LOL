# HeroesApp
Proyecto que tiene como fin hacer un CRUD y aprender mas sobre rutas, servicios, pipes y guards para las rutas en Angular
## Dev 
1. Clonar proyecto
2. Ejecutar ```npm i ```
3. Levantar el backend ```npm run backend ```
4. Ejecutar la app ```ng serve ```
